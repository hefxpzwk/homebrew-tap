# Hefxpzwk Tap

## How do I install these formulae?

`brew install hefxpzwk/tap/<formula>`

Or `brew tap hefxpzwk/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "hefxpzwk/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
